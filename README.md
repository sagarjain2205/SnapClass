# 📌 Overview

SnapClass is an **AI-powered smart attendance management system** that automates classroom attendance using:

- 🎭 Face Recognition
- 🎤 Voice Recognition
- 📷 Classroom Image Analysis
- 🧠 AI-based Student Detection

The platform provides separate dashboards for:

- 👨‍🎓 Students
- 👩‍🏫 Teachers

making attendance tracking fast, interactive, and intelligent.

---

# ✨ Features

## 👨‍🎓 Student Features

### 🔐 Face-Based Authentication

- Students register using their face
- AI generates facial embeddings
- Secure face-based login system

### 🎤 Voice Enrollment

- Optional voice profile creation
- Used for AI voice attendance

### 📚 Subject Enrollment

Students can enroll into subjects using:

- QR Codes
- Subject Codes
- Direct Join Links

### 📊 Student Dashboard

- View enrolled subjects
- Attendance tracking
- Subject statistics
- Unenroll from subjects

---

## 👩‍🏫 Teacher Features

### 📋 Interactive Teacher Dashboard

- Create multiple subjects
- Manage classroom subjects
- Share enrollment links instantly

### 📷 AI Face Attendance

Teachers can:

- Upload classroom images
- Detect multiple students at once
- Automatically mark attendance

### 🎤 AI Voice Attendance

Teachers can:

- Upload classroom voice recordings
- Detect speakers using AI
- Mark attendance automatically

### 📈 Attendance Management

- Attendance history storage
- Attendance analytics
- Present/Absent reports
- CSV export support

# 📷 Attendance Workflow

## Face Attendance Flow

1. Teacher uploads classroom images
2. AI scans student faces
3. Registered students are identified
4. Attendance marked automatically
5. Records stored in database

---

## Voice Attendance Flow

1. Teacher uploads classroom audio
2. AI extracts voice embeddings
3. Speaker matching performed
4. Attendance marked automatically
